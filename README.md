# Contributing

1. Fork the repo

2. Clone your forked repo

```
$ git clone https://github.com/<yourName>/CACS151.git
```

3. Make some changes as desired.

4. Commit changes

```
$ git add .
$ git commit -m "describe what you changed"
$ git push origin main
```

5. Open pull request
